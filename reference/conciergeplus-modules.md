# ConciergePlus — Full Module List
## AI-Driven Resident Portal (Part of Camelot OS)

| Module | Description |
|---|---|
| Amenity Booking | Book service elevator, party room, guest suites, BBQs. Calendar-managed, payment processing. |
| Announcements | Reach residents via email, SMS, or phone call. Emergency or routine updates. |
| Bikes | Manage bike spots, assign to units. |
| Classified Ads | Residents post items for trade, sale, or offer with photos. |
| Community Schedule | Track dates/appointments for owners, residents, staff groups. |
| Deliveries | Digital package management. Auto-alerts via email/SMS/phone. Barcode scanner integration. |
| Discussion Forum | Threaded discussions with approval controls. Every post tracked to a resident. |
| Entry Instructions | Permission-to-enter management. Key access grants + guest arrival notifications. |
| Equipment | Track HVAC, tools, systems — model numbers, serial numbers, purchase dates, maintenance logs. |
| Estoppel Certificates | Request and pay for condo estoppel certificates online. Requires Online Payments. |
| Incident Reports | Digital security incidents with photo/document attachments. Permanent record database. |
| Keys | Track all keys — what they're for, who has them. Digital sign in/out with full audit trail. |
| Lockers | Create, assign, manage storage lockers by unit. |
| Newsletter | Digital newsletter distribution with customizable features and targeted recipient lists. |
| Online Payments | Credit card payments for amenities. Funds deposited directly to corporation's bank account. |
| Parking | Create, assign, manage parking spots by unit. |
| Pass On Log | Security/manager shift handoff notes. Accountability tracking on who read what. |
| Pets | Pet registry — residents register pets, public-facing option for community awareness. |
| Quick Votes | Community polls with immediate results. Resident participation encouraged. |
| Rent/HOA Fee Processing | Batch recurring payments. Requires accounting system + payment gateway integration. |
| Residents' Guide | Searchable, printable digital resident guide. |
| Service Requests | Submit maintenance issues. Internal team communication on same ticket. Full tracking. |
| Tasks | To-do lists for PM, staff, anyone. Progress tracking. Board meeting reporting. |
| Urgent Announcements | <250 char quick alerts on homepage + digital displays if available. |
| Vacancy Dates | Track unit vacancy periods (seasonal, vacation). Affects deliveries, service requests. |
| Visitor Parking | Visitor parking management with pass printing and self-issue option. |
